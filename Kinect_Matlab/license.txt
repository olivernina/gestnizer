Kinect_Matlab.

Creator: Andrew Mendez
This zip-file contains c++ wrapper functions for the Microsoft Kinect, OpenNI libary.

Also includes additional programs:
SK_Recorder.m,Gesture_Data.m,and dataplot.m

This can run on either Windows or MacOSX_64_bit.
